# Vastu The Science - Astrologer Portfolio

A premium, modern portfolio website for an astrologer built with **Next.js**, **Tailwind CSS**, and **Framer Motion**. It features a "Celestial Chart" aesthetic with a solar system background, rotating text, and a responsive design.

## Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

- **Node.js**: Version 18.x or higher
- **npm**: Version 9.x or higher

### Local Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/vaibhav-sharma-2004/vastu-the-science.git
   cd Vastu-app
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up environment variables:**
   Create a `.env.local` file in the root directory and add the following variables:
   ```env
   # Resend API Key for contact form
   RESEND_API_KEY=your_resend_api_key
   
   # Email to receive contact form submissions
   CONTACT_RECEIVER_EMAIL=your_email@example.com
   
   # Public phone number for WhatsApp integration
   NEXT_PUBLIC_PHONE_NUMBER=your_phone_number
   
   # Default WhatsApp message
   NEXT_PUBLIC_MESSAGE="Hi! I'm interested in a consultation."
   ```

4. **Run the development server:**
   ```bash
   npm run dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
